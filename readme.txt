An easy way to show that you will be attending this year's ZendCon conference.

== Description ==

If you are going to be a speaker, attendee, sponsor or exhibitor at ZendCon this year
you can use this plugin to create a really simple widget on your blog that states how
you will be attending.

== Installation ==

* Copy the zendcon-badges directory into your wp-content/plugins directory
* Activate it
* Go to "Settings" -> "ZendCon"
* Set how you will be participating and the size of the image you want
* Go to "Appearance" and drag the ZendCon Badges widget to wherever you want it on the page
